# app-frontend-apereira
app para prueba tecnica

1- Descargar proyecto app-frontend-apereira
2- Agregar la carpera del proyecto y ubicar en un directorio 
3- Desde el terminar entrar en la carpeta raiz del proyecto
4- Ejecutar el comando npm install para instalar dependencias
5- Ejecutar el comando ng serve --open para ver el proyecto en funcionamiento desde el navegador
6- Poner en marcha el servicio backend con el comando npm run start:dev

Nota:
En el servicio repo-interview-main debe habilitar cors en el archivo main.ts (descomentar linea 8)   ya que de no hacerlo, desde el frontend presentara un bloqueo por proxy en el consumo de la API
